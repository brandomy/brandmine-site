# 🤖 Claude / ChatGPT Quick Prompt Guide for Brandmine

Use this at the start of a conversation:

> I'm working on a multilingual Jekyll site (Brandmine) using `/en`, `/ru`, `/zh` subfolders. Brands and tags are Markdown files. Tags include `sectors`, `attributes`, `signals`, and `markets`. UI labels and nav data are in `_data/translations/` and `_data/navigation/`. Design tokens define fonts, colors, spacing in `tokens.css`. Use `_includes/` and `_layouts/` for rendering.

Then say what you’re trying to do:
> “Help me debug the `related-brands-list.html` on tag pages.”
