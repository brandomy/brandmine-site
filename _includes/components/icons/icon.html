{% comment %} Include Name: components/icons/icon.html Purpose: A flexible component for displaying PNG icons from the Brandmine asset library. Icons are loaded based on type (sector, market, attribute, signal, insight), name, and size. Parameters: - type: The icon type - one of 'sector', 'market', 'attribute', 'signal', or 'insight' (required) - name: The specific icon identifier (e.g., 'hotels-resorts', 'founder-led') (required) - size: Pixel size - one of '24', '48' (optional, default: '24') Dependencies: - PNG icon files in assets/images/icons/dimensions/{type}/{size}/{name}.png - CSS: Styles for .icon and icon modifiers (e.g., .icon--sector, .icon--24) Example Usage: {% include components/icons/icon.html type="sector" name="hotels-resorts" %} {% include components/icons/icon.html type="attribute" name="founder-led" size="48" %} {% include components/icons/icon.html type="market" name="russia" size="24" %} Notes: - Icons fallback to default image if the combination of name/type is invalid - Asset structure must match the path: assets/images/icons/dimensions/{type}/{size}/{name}.png {% endcomment %} {% assign icon_type = include.type | default: 'sector' %} {% assign icon_name = include.name | default: 'sector-default' %} {% assign icon_size = include.size | default: '24' %} {% assign icon_class = 'icon icon--' | append: icon_type | append: ' icon--' | append: icon_size %} {% assign icon_src = '/assets/images/icons/dimensions/' | append: icon_type | append: '/' | append: icon_size | append: '/' | append: icon_name | append: '.png' %}
