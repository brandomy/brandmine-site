{% comment %} Component: helpers/country-name.html Purpose: Converts ISO country codes to human-readable country names in the appropriate display language. Parameters: - code: (required) The 2-letter ISO country code to look up (e.g., "ru", "br", "cn") - lang: (optional) The display language for the name. Defaults to current page language Dependencies: - _data/countries_basic.json: Contains country code mappings with translations Example Usage: {% include components/helpers/country-name.html code="ru" %} Returns: "Russia" (if page.lang is "en") {% include components/helpers/country-name.html code="ru" lang="zh" %} Returns: "俄罗斯" (explicitly set to Chinese) Fallback Behavior: - If the country code isn't found in the JSON, returns the code itself - If the requested display language isn't available, returns the code {% endcomment %} {% assign country_code = include.code %} {% assign display_lang = include.lang | default: page.lang %} {% assign country_info = site.data.countries.countries[country_code] %} {% if country_info and country_info.name and country_info.name[display_lang] %}{{ country_info.name[display_lang] }}{% else %}{{ country_code | upcase }}{% endif %}
